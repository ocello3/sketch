## 初回ループの判定
最初はobjsがundefinedかどうかで、初回ループの判定を行っていたが、毎ループオブジェクト全体の判定を行うのは非効率に思えた。
そのため、初期化時にisInitのみ定義して、初回ループの最後のisInitを更新する運用とした。
```js
const obj = { isInit: true };

const update obj = (obj) => {
    // 処理
    {
        // other properties
        isInit: false;
    }
    return isInit;
}
```
